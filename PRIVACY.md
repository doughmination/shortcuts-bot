# Privacy Policy

**Last Updated: December 15, 2025**

## Data Collection
This bot collects minimal data necessary for operation:
- User IDs (when mentioned in commands)
- Command usage (for functionality)

## Data Usage
Collected data is used solely to:
- Execute requested commands
- Provide bot functionality

## Data Storage
- No persistent user data is stored
- Interaction data is processed in memory and not retained

## Data Sharing
We do not sell, trade, or share user data with third parties.

## Discord's Privacy Policy
This bot operates on Discord's platform and is subject to [Discord's Privacy Policy](https://discord.com/privacy).

## User Rights
Since we don't store personal data, there is no data to request or delete.

## Changes
This policy may be updated at any time. Check this page for the latest version.

## Contact
For privacy concerns, contact the bot owner through Discord.
